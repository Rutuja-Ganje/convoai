# ConvoAI 🤖

An AI-powered conversational platform built with the MERN stack, designed to deliver intelligent, real-time chat experiences through a clean and modern interface.

## 🛠️ Tech Stack

- React
- Node.js
- Express.js
- MongoDB

## ✨ Features

- User Authentication
- AI-Powered Conversations
- Real-Time Chat Interface
- Chat History
- Responsive UI
- Secure Backend APIs
